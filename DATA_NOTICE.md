# Data notice

This repository contains a structured extraction of selected nutrition table pages from:
- `Buku Tabel Komposisi Pangan Indonesia tahun 2020`

## Important
- This repository does **not** claim new ownership over the underlying source data.
- Public availability of this repository does **not** replace checking the original publication, publisher, or government source terms.
- For official context, methodology, and provenance, consult the original publication.

## Why this repository exists
The goal is to make the selected table section easier to:
- search
- audit
- load into software
- compare across CSV / JSON / SQLite formats
